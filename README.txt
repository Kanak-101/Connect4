Set up JavaFX environment
In IntelliJ IDEA:
Go to Run > Edit Configurations.
Add the --module-path and --add-modules options in the VM options.
eg.: --module-path "K:\Drive\Java Course\JavaFX App\javafx-sdk-17.0.17\lib"
--add-modules=javafx.base,javafx.controls,javafx.graphics,javafx.media,javafx.fxml
